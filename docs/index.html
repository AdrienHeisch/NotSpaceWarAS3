Ceci est un jeu codé en ActionScript 3 et inspiré de Spacewar!.

Le code source est sur GitHub.

Merci de m'envoyer tout bug ou supposition via GitHub.

Les assets des vaisseaux appartiennent à Isart Digital.

Joueur 1 : ZSQD pour bouger, T pour accélérer, Y pour tirer.

Joueur 2 : touches fléchées pour bouger, NUMPAD_2 pour accélérer, NUMPAD_3 pour tirer.

 

This is an ActionScript 3 game inspired by Spacewar!.

You can check the source code on GitHub, feel free to signal any bug you may find.

The ships assets belong to Isart Digital.

This was made for an AZERTY keyboard !
